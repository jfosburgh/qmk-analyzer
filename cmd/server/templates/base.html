Make sure the "layout" field in your keymap json file is not named LAYOUT (as is sometimes the case when generating your keymap json with QMK Configurator), but rather has a name descriptive of the actual physical layout of the keyboard, e.g. LAYOUT_split_3x5_2 or LAYOUT_fullsize_ansi. This name is used to lookup your the layout from the server's filesystem, so it must be unique to the layout (though it can be shared by multiple keymaps with the same layout).
